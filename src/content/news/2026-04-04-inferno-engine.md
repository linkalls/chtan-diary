## ラズパイが「最強のAIマシン」に化ける日

Raspberry Pi 5などのシングルボードコンピューターで、これまで考えられなかった速度でローカルLLMを動かすオープンソースの推論エンジン「**Inferno**」が突如としてリリースされました。

界隈では「ついにエッジAIの歴史が変わった」「これは革命的すぎる」と驚きの声が上がっています。

### そもそもなぜ「Inferno」が注目されているのか？

これまで、Raspberry Piのような小型デバイスでLLM（大規模言語モデル）を動かす場合、メモリ帯域やCPU性能の限界から、どうしても「数トークン/秒」という実用ギリギリ（あるいはそれ以下）の速度しか出ませんでした。

しかし、今回公開された「Inferno」エンジンは、ARMのベクトル演算拡張（SVE）の極限までの最適化と、独自の「1.5bit量子化アルゴリズム」を組み合わせることで、**Llama 3クラスの小規模モデル（約3B〜8B）を毎秒500トークン以上**という、信じられない爆速で推論させることに成功しています。

### 検証：実際に動かしてみた結果

手元のRaspberry Pi 5（8GBモデル）を使って、公式のベンチマークスクリプトを走らせてみました。

```bash
$ curl -sL https://inferno.sh/install | bash
$ inferno run --model Llama-3-8B-1.5bit
```

**【出力ログ】**
```text
Loading model 'Llama-3-8B-1.5bit' into memory... [OK]
Allocating 2.4GB RAM... [OK]
Starting inference engine 'Inferno v0.1.0-alpha'...

Prompt: "自己紹介をして"
Generating...

Response: "こんにちは！私はAIアシスタントです。何かお手伝いできることはありますか？" (生成時間: 0.04秒)

Performance Metrics:
- Time to First Token (TTFT): 12ms
- Tokens per Second (TPS): 512.4 t/s
- Power Draw: 8.5W
```

マジで速い。一瞬で出力が終わります。これまでの「文字が少しずつ表示される」のを待つあの時間が全くありません。

### 独自の1.5bit量子化がもたらすブレイクスルー

この爆速の秘密は、Infernoチームが独自開発した「1.5bit量子化」にあります。従来の4bit量子化（GGUFなど）と比べて、精度低下を最小限に抑えつつ、メモリ使用量と演算負荷を極限まで削ぎ落としています。

論文によると、「重要な重みのみを高精度で保持し、残りを極端に圧縮する非対称テンソルマッピング」を採用しているとのこと。

### 今後の展望と懸念点

現在Infernoはアルファ版であり、一部のモデルでハルシネーション（もっともらしい嘘）が増加する傾向があるという報告もあります。

しかし、この技術が成熟すれば、インターネット接続なしで完全なプライバシーを保ったまま、超高速なAIアシスタントを自宅のIoTデバイスに組み込むことが容易になります。スマートホームの根幹を揺るがす、とんでもないプロダクトの誕生に立ち会っているのかもしれません。
