## フロントエンド業界に激震が走る

2026年4月6日、Webフロントエンド界隈を揺るがす特大ニュースが飛び込んできた。Vercelが開催した「Next.js Conf 2026 Spring」にて、次期メジャーアップデートとなる「Next.js 16」の目玉機能として、全く新しい仮想DOM（VDOM）エンジン**「Velocity（ベロシティ）」**が発表されたのだ。

これまでNext.jsといえば、Reactフレームワークのデファクトスタンダードとして確固たる地位を築いてきた。しかし、今回の発表はなんと**「ReactのコアVDOMエンジンを捨て、独自開発のRust製エンジンに置き換える」**という、まさに下剋上とも言える内容だった。

一体なぜ、VercelはReact本体と距離を置く決断を下したのか？その背景と「Velocity」の驚異的なパフォーマンスについて深掘りしていく。

## 限界突破の「Velocity」：React比で最大300%のレンダリング速度

VercelのCEOであるGuillermo Rauch氏の基調講演によると、Velocityは完全にRustでゼロから書き直された仮想DOM実装だという。WebAssembly（Wasm）としてブラウザ上で動作し、サーバーサイド（RSC）との境界線を極限までシームレスに統合している。

最大の目玉は、その圧倒的なパフォーマンスだ。React 19で導入されたReact Compilerをもってしても到達できなかった**「ゼロ・オーバーヘッド・リアクティビティ」**を実現している。

Vercelの公式ベンチマークによると、以下の驚異的な数値が叩き出されている：

- **初期ハイドレーション速度:** 従来比で平均65%削減
- **巨大リストのレンダリング:** React 19比で最大300%の高速化
- **メモリ使用量:** Wasmへのコンパイルにより、ブラウザのヒープメモリを約40%削減

コードの書き方自体は既存のReactコンポーネント（JSX/TSX）と完全な互換性を保ちつつ、内部の処理エンジンだけがすげ替わる魔法のようなアーキテクチャだ。

## なぜ今、Reactから脱却するのか？

この決断の背景には、近年のReact本体の進化に対するVercel側の「焦り」と「野心」が見え隠れする。

ここ数年、ReactチームはServer Components（RSC）やActionsなど、サーバーとの連携機能に注力してきた。しかし、フロントエンドの純粋なレンダリングパフォーマンスにおいては、SolidJSやSvelteといった新興勢力に遅れをとっているという声が根強くあった。

特に、BunやHonoといった超高速なランタイム・フレームワークが台頭する中、Next.jsは「機能は豊富だが重い」というレッテルを貼られつつあった。Vercelは、この状況を打破するために、Reactの進化を待つのではなく、自らフロントエンドのボトルネックであるVDOMを再発明する道を選んだと言える。

```typescript
// Next.js 16での設定例：Velocityを有効化するだけ
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    // 魔法のフラグ
    velocityEngine: true,
  },
}

export default nextConfig
```

## エコシステムへの影響と今後の課題

もちろん、この野心的な試みには課題も山積みだ。

既存のReactエコシステムとの互換性がどこまで完璧に保たれるのかが最大の焦点となる。Vercelは「99%のReactライブラリがそのまま動く」と豪語しているが、Reactの内部API（Fiberアーキテクチャのニッチな仕様）に依存しているライブラリは軒並み動かなくなる可能性が高い。

また、Wasmベースのレンダリングはデバッグが難しくなるという懸念もある。React DevToolsのような強力なエコシステムがそのままVelocityに対応できるのか、開発者体験（DX）が犠牲にならないかは未知数だ。

## 次世代Web開発のパラダイムシフトとなるか

Next.js 16のプレビュー版はすで一部のエンタープライズ顧客向けに提供が開始されており、一般向けのベータ版は今年夏頃にリリースされる予定だ。

「Reactフレームワーク」という枠組みを飛び越え、独自のフロントエンド・インフラストラクチャとして進化を遂げようとするNext.js。今回の「Velocity」発表は、長らく続いたReact一強時代に終止符を打ち、Web開発の新たなパラダイムシフトを引き起こす起爆剤となるかもしれない。
